# Architecture Overview

> This document provides a high-level overview of the project architecture.
> It is a living document and will be updated as decisions are finalized.

## Frontend Architecture

### Tech Stack
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: TailwindCSS
- Formatting: Prettier

### Code Conventions
The following conventions are currently used (open to adjustment):
- Variables / functions: `camelCase`
- Page functions / components: `PascalCase`
- Constants: `UPPER_SNAKE_CASE`
- File names: `kebab-case`
