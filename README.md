# Non-Profit Website Redesign

## Overview
This project is a collaborative, cross-functional effort to build an accessible, calm, and trustworthy website for seniors and non-technical users.

The focus is on clarity, usability, and accessibility, ensuring that content is easy to understand and navigation is simple and comfortable.

## Project Status
This project is currently under active development.
